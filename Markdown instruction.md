## Списки
Чтобы добавить не нумернованные списки, необходимо пункты выделить звёздочкой (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить нумерованные списки, небходимо пункты просто пронумеровать. 
Напрмиер, вот так: 
1. Первый пункт
2. Второй пункт
