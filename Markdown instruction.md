# Инструкция для работы с Markdown

## Выделение ткста
Что бы выделить текст курством необходимо обрамить его звёздочками (*). Например, *вот так*.

Чтобы выделить текст полужирным, необходимо обрамит ео двойными звёздочками (**).
Например, **вот так**
